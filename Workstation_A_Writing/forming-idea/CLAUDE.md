*(EXAMPLE — the Forming / stub case.)* A new idea that has just started — too little motion has run to tell
what it will **become** (a Reasoning Workspace? a Reference Repo? a Build?). So it earns almost no structure

## STATE GATES STRUCTURE
The rule at its starting point: **structure is earned by Active state, not created up front.** A stub is the
deliberate *absence* of the four-home model — no `references/`, no `_reasoning-log/`, no wiki, no `stages/`.
When the idea runs enough motion to show its kind, it promotes into that kind's structure *then* — not now.
